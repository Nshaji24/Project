CONGRATS!

Now that you have your PC you might be confused as to do first. Theres so many things you can do with a PC so I'll list a few things that will help you get started

USEFUL SOFTWARE

  • Steam:Where you get the latest games, Steam has a huge collection of games
  • Spotify:Listening to music while you work and game makes it much more enjoyable
  • Security: Tons of options out there, research around and find one you like
  • MSI Afterburner: Lets you monitor temperatures of PC parts to see if anything is wrong
  • Not software but a quality mice and keyboard are essential

Whats next?

  • Clean out bloatware (Pre-Installed Apps that come with windows that you dont need
  • Install/Update latest drivers
  • Run some benchmark tests
  • Find some games you like and get to gaming!
  • Plan out parts you might want to upgrade
